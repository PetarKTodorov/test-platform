## Initial setup

### Project setup
 ```shell
nvm install
```

### Starting the project locally
```shell
nvm use
npm install
```

## Front-End

1. Navigate to "Webpack" folder
2. Run one of the  following commands:
    * `npm run build-prod`
    * `npm run build-dev`
    * `npm run watch`
    * `npm run stats`
